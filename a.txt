# Code Translation: Java ↔ C# using CodeBERT

**Objective:**  
Produce CodeXGLUE Java ↔ C# code translation results using the provided dataset and baseline CodeBERT model.  
We aim to fine-tune the model, generate translations, and evaluate BLEU, accuracy & performance metrics.

**Dataset:**  
- Source: Public repos (Lucene, POI, JGit, Antlr)  
- Format: Parallel functions in Java & C#  
- Split: Train (10,300), Valid (500), Test (1,000)

**Model:**  
- Baseline: CodeBERT (RoBERTa-based pretrained model)  
- Task: Seq2Seq code translation

import numpy as np
import seaborn as sns

def similarity(a, b):
    a_set = set(a.split())
    b_set = set(b.split())
    return len(a_set & b_set) / max(len(a_set), 1)

sample_refs = refs[:20]
sample_preds = preds[:20]

matrix = np.zeros((20, 20))

for i in range(20):
    for j in range(20):
        matrix[i][j] = similarity(sample_refs[i], sample_preds[j])

plt.figure(figsize=(8,6))
sns.heatmap(matrix)
plt.title("Prediction vs Reference Similarity Heatmap")
